# ImGuiReduxFont

为GTA游戏提供完整自定义字体支持的ImGui界面库

## 代码示例

```javascript
// 启用自定义字体支持
ImGui.SetCustomFontEnabled(true);

// 检查是否启用了自定义字体支持
if (ImGui.IsCustomFontEnabled()) {
    // 加载自定义字体
    ImGui.LoadCustomFont("assets/AlibabaPuHuiTi-Black.ttf", 16.0);
}
```
